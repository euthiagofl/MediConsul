# MediConsul
Criação de um wireframe de média fidelidade.
link do figma: https://www.figma.com/make/W0kkIxrWqIiYtuKRfeowPm/MediConsul-appointment-app?fullscreen=1&t=3rdQW8Ko9UDswlBV-1&preview-route=%2Flogin

Princípios de UX aplicados:
  Lei de Hick:
    Poucas opções por tela para facilitar decisão.
  Lei de Fitts:
    Botão Confirmar consulta grande e central.
  Consistência:
    Botões e layout iguais em todas as telas.
