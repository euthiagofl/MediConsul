# MediConsul
Criação de um wireframe de média fidelidade.
link do figma: https://www.figma.com/make/W0kkIxrWqIiYtuKRfeowPm/MediConsul-appointment-app?fullscreen=1&t=3rdQW8Ko9UDswlBV-1&preview-route=%2Flogin

Princípios de UX aplicados:
  Lei de Hick:
    Poucas opções por tela para facilitar decisão.
  Lei de Fitts:
    Botão Confirmar consulta grande e central.
  Consistência:
    Botões e layout iguais em todas as telas.
    Critério estudado por Christian Bastien e Dominique Scapin.

    a.png: página de login para usuários já cadastrados e com opção para cadastrar novos usuários
    b.png: página de cadastro para novos usuários
    c.png: escolher entre agendar nova consulta ou verificar consultas já marcadas ou canceladas
    d.png: escolher especialidade médica
    e.pgn: escolher médico
    f.pgn: escolher data e hora da consulta de acordo com a disponibilidade no sistema
    g.png: verificação das consultad já agendadas ou canceladas (caso tenha cancelado) e com opção para cancelar consulta já marcada ou  marcar nova consulta
